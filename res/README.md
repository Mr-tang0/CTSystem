# 资源文件说明

此文件夹存放操作文档所需的图片资源。

## 目录结构

```
res/
├── images/                    # 图片资源
│   ├── detector_connect.png   # 探测器连接界面截图
│   ├── detector_panel.png     # 探测器面板截图
│   ├── hvps_connect.png       # 高压电源连接截图
│   ├── raysource_connect.png  # 射线源连接截图
│   ├── motor_connect.png      # 位移台连接截图
│   └── motor_control.png      # 位移台控制截图
└── README.md                  # 本说明文件
```

## 图片要求

| 文件名 | 尺寸建议 | 内容说明 |
|--------|---------|---------|
| detector_connect.png | 800x600 | 探测器设备连接界面 |
| detector_panel.png | 800x600 | 探测器控制面板和监控面板 |
| hvps_connect.png | 800x600 | 高压电源连接和设置界面 |
| raysource_connect.png | 800x600 | 射线源连接和状态界面 |
| motor_connect.png | 800x600 | 位移台连接界面 |
| motor_control.png | 800x600 | 位移台控制面板 |

## 添加图片

1. 截取对应功能的界面截图
2. 按上述命名规则保存到 `images` 文件夹
3. 更新操作文档中的图片引用

## 图片格式

- 格式：PNG（推荐）或 JPG
- 背景：建议使用浅色背景
- 标注：可添加必要的箭头和文字说明
